# Custom PC Build for IT Labs

## Overview

I built a custom PC to support my IT learning, including virtualization, networking labs, and system administration practice.

## Hardware Specifications

* **Case:** MUSETEX ATX Form Factor
* **CPU (Processor):** AMD Ryzen 5 5500 (6 Cores / 12 Threads, up to 4.2 GHz)
* **GPU (Graphics Card):** SOYO AMD Radeon RX 580 8GB GDDR5
* **Motherboard:** MSI B550 Gaming Plus
* **RAM (Memory):** 16GB DDR4 3200MHz (2x8GB CL16)
* **Storage:** 256GB NVMe SSD (3200 MB/s)
* **Power Supply:** Thermaltake Smart 700W 80+
* **Thermal Paste:** Arctic MX-6

## Purpose

This system is used for:

* Running virtual machines (VirtualBox)
* Practicing IT support and troubleshooting
* Learning networking and system administration

## What I Learned

* How to assemble a computer from scratch
* Understanding hardware compatibility
* Importance of cooling and power supply
* Installing and configuring system components

## Future Plans

* Install VirtualBox and create virtual machines
* Build a home lab environment
* Practice networking between systems
* Simulate real IT troubleshooting scenarios
